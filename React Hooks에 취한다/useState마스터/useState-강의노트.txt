# useState 훅 알아보기

=> 용어 알아보기
    01. state : 컴포넌트의 상태를 의미한다. (== 현재 상태 값)
    02. useState : 컴포넌트의 상태를 간편하게 생성하고
                    업데이트 하는 도구를 제공해준다.

=> const[state, setState] = useState(초기값);
    01. state
        : 현재의 상태 값을 의미한다.

    02. setState(함수)
        : 현재의 상태를 변경해주고 싶을 때 사용한다.
        : 함수 안에 인자로 변경될 값을 넣어준다.
        : 해당 컴포넌트는 다시 업데이트(=렌더링)된다.    
        
    03. 초기값
    04. useState
